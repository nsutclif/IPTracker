Free, No Ads

This web application was created by Nathan Sutcliffe as a professional development project. The code was built to scale up, but the pricing model (free, no ads) was not! If people actually use this thing, the availability and/or pricing model may change.

Beta

At the present time, you can consider this web application to be in beta. As I continue to tinker with it, you can expect that some bad things may happen:

  • There may be unannounced service outages
  • Your data may go missing
  • Features may appear and disappear

Built with the MEAN.JS template, starring:

  • MongoDB
  • Express
  • Angular
  • Node.js

Also featuring:

  • Passport
  • Nodemailer
